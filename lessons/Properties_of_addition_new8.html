Sort each number sentence into the property it shows.

  • 14 + 6 = 6 + 14
  • (2 + 7) + 5 = 2 + (7 + 5)
  • 38 + 0 = 38
  • 9 + (1 + 4) = (9 + 1) + 4
  • 25 + 30 = 30 + 25
  • 0 + 61 = 61

Swapped order is commutative, new brackets is associative, and adding 0 is identity.

Priya wants to add 47 + 3 + 20. She rewrites it as (47 + 3) + 20 so she can add 47 + 3 = 50 first. Which property lets her do this?

  • Associative Property
  • Commutative Property
  • Identity Property

She only changed the grouping, not the order — that's the associative property.

  • Adding 0 to a number leaves it unchanged.
  • 0 + 94 = 94
  • It lets you change the order of the addends.
