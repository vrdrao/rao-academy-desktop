## After that first message

## The two things to watch for

**1. Your old lesson files may have `<style>` blocks baked in.**
They were generated by the old engine, which stapled CSS onto every question. They'll
still work, but they fight `rao.css`. `npm test` will surface it. There may be a
one-time cleanup — ask Claude Code to script it across all files at once.

**2. A RED test is a GOOD outcome.**
It means we found a genuinely missing capability — via a failing test, not via a child
seeing a blank box. That's exactly what you're trying to discover.
